# Python Development Environment Setup

1. **Python Installed**:
   - Python 3.12 has been installed to `C:\Users\Shin-Nyum\AppData\Local\Programs\Python\Python312\python.exe`.

2. **Virtual Environment**:
   - A virtual environment `venv` has been created in this folder.
   - To activate it in PowerShell: `.\venv\Scripts\Activate.ps1`
   - To activate it in CMD: `.\venv\Scripts\activate.bat`

3. **Run your code**:
   - After activation, simply run: `python main.py`
   - Or run directly using the venv python: `.\venv\Scripts\python.exe main.py`
